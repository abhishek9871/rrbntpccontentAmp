Mensuration - List of Formulas in Elementary Geometry

Overview

This page provides a comprehensive list of formulas in elementary geometry, covering area and perimeter for two-dimensional shapes and volume for three-dimensional shapes.

Two-Dimensional Shapes

Square

l² area, 4l perimeter

Rectangle

lb area, 2(l+b) perimeter

Square

  • Area: l²
  • Perimeter: 4l
  • Symbols: l = length of a side

Rectangle

  • Area: lb
  • Perimeter: 2(l+b)
  • Symbols: l = length, b = breadth

Circle

  • Area: πr²
  • Perimeter: 2πr or πd
  • Symbols: r = radius, d = diameter

Ellipse

  • Area: πab
  • Perimeter: No simple formula
  • Symbols: a = semimajor axis, b = semiminor axis

Triangle

  • Area: bh/2
  • Perimeter: a+b+c
  • Symbols: b = base, h = height, a,b,c = sides

Parallelogram

  • Area: bh
  • Perimeter: 2(a+b)
  • Symbols: b = base, h = height, a = side

Trapezoid

  • Area: (a+b)/2 × h
  • Perimeter: No simple formula
  • Symbols: a,b = bases

Three-Dimensional Shapes

Sphere

Volume: (4/3)πr³
Surface Area: 4πr²

Cone

Volume: (1/3)πr²h

Cylinder

Volume: πr²h

Cube

Volume: a³

Cuboid

Volume: abc

Prism

Volume: Bh

Pyramid

Volume: (1/3)Bh

Tetrahedron

Volume: (√2/12)a³

Parallelepiped

Volume: abc√(K)

Ellipsoid

Volume: (4/3)πabc

Cone

  • Volume: (1/3)πr²h
  • Symbols: r = base's radius, h = cone's height

Cube

  • Volume: a³
  • Symbols: a = side's length

Cuboid

  • Volume: abc
  • Symbols: a, b, c = sides' length

Cylinder

  • Volume: πr²h
  • Symbols: r = base's radius, h = cylinder's height

Ellipsoid

  • Volume: (4/3)πabc
  • Symbols: a, b, c = semi-major and semi-minor axes' length

Sphere

  • Volume: (4/3)πr³
  • Surface Area: 4πr²
  • Circumference: 2πr
  • Radius Formulas: C/(2π), √(S/(4π)), ³√((3V)/(4π))
  • Symbols: r = radius, C = circumference (of a great circle), S = surface area, V = volume

Parallelepiped

  • Volume: abc√(K)
  • Symbols: a, b, c = sides' length, K = 1 + 2cos(α)cos(β)cos(γ) - cos²(α) - cos²(β) - cos²(γ), α, β, γ = angles between the two sides

Prism

  • Volume: Bh
  • Symbols: B = base's area, h = prism's height

Pyramid

  • Volume: (1/3)Bh
  • Symbols: B = base's area, h = pyramid's height

Tetrahedron

  • Volume: (√2/12)a³
  • Symbols: a = side's length
