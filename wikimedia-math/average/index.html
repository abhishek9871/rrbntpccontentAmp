Overview

In statistics and mathematics, averages are measures of central tendency that represent a typical value for a dataset. The three primary measures of central tendency are mean, median, and mode.

Mean (Arithmetic Average)

Definition

The mean or arithmetic mean is the sum of all values divided by the number of values.

Formula

Mean (x̄) = (x₁ + x₂ + x₃ + ... + xₙ) / n

or

x̄ = Σx / n

where Σ means "sum of" and n is the number of values

Example

Test scores: 81, 84, 93

Mean = (81 + 84 + 93) / 3 = 86

Properties of Mean

  • Always lies between the highest and lowest values
  • Affected by extreme values (outliers)
  • Used for symmetric distributions
  • Sum of deviations from mean is always zero

Median

Definition

The median is the middle value when values are arranged in order.

Calculation

  • Odd number of values: Middle value
  • Even number of values: Average of two middle values

Examples

Odd number of values: Test scores: 81, 84, 93

Ordered: 81, 84, 93

Median = 84

Even number of values: Test scores: 81, 84, 86, 93

Ordered: 81, 84, 86, 93

Median = (84 + 86) / 2 = 85

Properties of Median

  • Not affected by extreme values
  • Better for skewed distributions
  • Used for ordinal data

Mode

Definition

The mode is the value that occurs most frequently in a dataset.

Examples

Single mode: Test scores: 81, 84, 84, 93

Mode = 84

Multiple modes: Test scores: 81, 84, 84, 90, 90

Modes = 84 and 90

No mode: Test scores: 81, 84, 87, 90

No mode (all values appear once)

Properties of Mode

  • Can have no mode, one mode, or multiple modes
  • Not affected by extreme values
  • Useful for categorical data
  • May not exist or be unique

Weighted Average

Definition

A weighted average considers the importance (weight) of each value when calculating the average.

Formula

Weighted Average = (w₁x₁ + w₂x₂ + ... + wₙxₙ) / (w₁ + w₂ + ... + wₙ)

where w₁, w₂, ..., wₙ are the weights and x₁, x₂, ..., xₙ are the values

Example

Two classes with different sizes:

  • Morning class: Average = 80%, 20 students
  • Afternoon class: Average = 90%, 30 students

Weighted Average = (20×80 + 30×90) / (20 + 30) = 86%

Geometric Mean

Definition

The geometric mean is a number midway between two values by multiplication, rather than by addition.

Formula

Geometric Mean = √(A × B)

For two values A and B

For n values: Geometric Mean = ⁿ√(A₁ × A₂ × ... × Aₙ)

Example

Values: 3 and 12

Geometric Mean = √(3 × 12) = √36 = 6

(Because 3 × 2 = 6, and 6 × 2 = 12)

Comparison Table

Measure Definition Formula Best Used For
Mean Sum of values divided by count Σx / n Symmetric distributions
Median Middle value when ordered Middle value or average of middle two Skewed distributions
Mode Most frequent value Value with highest frequency Categorical data
Weighted Mean Average considering importance Σ(wx) / Σw Different levels of importance
Geometric Mean Multiplicative midpoint ⁿ√(product of values) Ratios, percentages

When to Use Each Measure

Use Mean When:

  • Data is normally distributed
  • No extreme outliers
  • You need a value that can be used in further calculations

Use Median When:

  • Data is skewed or has outliers
  • Working with ordinal data
  • Need a robust measure

Use Mode When:

  • Working with categorical data
  • Finding the most common category
  • Describing typical experience
